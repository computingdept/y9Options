Teacher Guidance – Lesson 3
Binary (CS) vs Excel (IT)

Overview

Approx. length: 60 minutes Year group: 9 Options Taster

Focus: Convert numbers between denary, 8-bit binary, and hex (CS), then build an Excel sheet to automate conversions (IT).

Learning Objectives

  • Convert 0–255 to 8-bit binary and to hex.
  • Explain place values (128–1) and nibble-to-hex groupings.
  • Create an Excel sheet that outputs 8-bit binary (and hex as an extension).

Lesson Flow (approx.)

Students complete the Do Now silently in Microsoft OneNote (Class Notebook) under “Lesson 3 – Do Now”. Use the Snipping Tool (Win + Shift + S) for any screenshots later in the lesson.
Approx. Time Stage Teacher Guidance (modelling optional) Student Outcome / Evidence
Start (≈ 5 min) Do Now – Silent recap Display the Do Now. Remind: quiet answers in OneNote. After a few mins, click Reveal and check a couple quickly. Written responses in OneNote under “Lesson 3 – Do Now”.
≈ 5 – 12 min Knowledge Recap Use the Knowledge boxes: bits/byte (0–255), base-2, and hex shortcut (group 4 bits). Use the “light switches” analogy for place values. Class can state why 8 bits cover 256 values and how hex groups work.
≈ 12 – 22 min CS – Conversions (direct practice) If helpful: Model one example on the board (e.g., 25 → 00011001 → 19). Prompt thinking: “Which highest place value fits? Subtract. Repeat.” 5 numbers converted to 8-bit binary and hex. Evidence: Paste a small table or a screenshot into OneNote under “Lesson 3 – CS Activity”.
≈ 22 – 30 min CS – Quick checks (online) Point to the two Bitesize pages if needed; then let students try 5 Qs on each Bag of Cows quiz for feedback. Evidence: Snip quiz feedback (score/result) into OneNote (optional if tight for time).
≈ 30 – 45 min IT – Excel converter If helpful: Model a single row: A2 = number → B2:I2 bit formulas (INT(MOD(...))) → TEXTJOIN in J2. Mention built-in =DEC2BIN(A2,8) as an alternative. Extension: two-nibble hex with INT(A2/16), MOD(A2,16) + lookup table. Working sheet that outputs 8-bit binary (and hex if extended). Evidence: Snip the sheet (inputs + outputs) into OneNote under “Lesson 3 – IT Activity”.
≈ 45 – 53 min Key Vocabulary (after practice) Reinforce: Binary, Bit, Byte, Hex. Ask pupils to point to where each appeared in their work today. Verbal recall; optional brief note in OneNote.
≈ 53 – 60 min Exit Ticket / Reflection Use the three questions from the student page. Responses in OneNote. Saved in Class Notebook under “Lesson 3 – Exit Ticket”.

Support & Extension

  • Support: Provide a place-value strip (128–1). Allow partner talk during conversions. Offer a starter Excel file with headings.
  • Extension: Add automatic hex output (two-nibble table) and validate A2 to 0–255 only. Try a conditional format to colour bits.
  • SEND: Keep fonts large and high contrast; model only if helpful; circulate with quick checks; ensure independent use of Snipping Tool for evidence.
