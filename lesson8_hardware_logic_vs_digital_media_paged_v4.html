What you'll learn today

By the end of this lesson, you will be able to:

  • Identify key hardware parts and what they do.
  • Explain simple logic gates (AND, OR, NOT).
  • Create a short digital media asset: image/audio/video.

How you'll know you've got it

  • Match hardware to its job (CPU, RAM, storage, GPU, I/O).
  • Fill in a 3‑gate truth table correctly.
  • Export a small media file with sensible settings.

Key Vocabulary

CPU — the brain that runs instructions.
RAM — fast memory for active work.
Storage — saves files when power is off.
GPU — draws images and speeds up media work.
I/O — input/output devices (keyboard, mouse, mic).
Logic gate — a small circuit that makes a decision.
Truth table — shows outputs for all inputs.
Resolution / Bit rate — quality settings for media.
