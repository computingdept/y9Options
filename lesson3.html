IT Activity ‚Äì Build an Excel Sheet: 0‚Äì255 ‚Üí 8-bit Binary (Extension: Hex)

Create a spreadsheet that converts a decimal number to an 8-bit binary string. Then add an extension using VLOOKUP to show the hex value.

  1. In cell A2, type a number from 0 to 255 (try lots!).
  2. In cells B2:I2, show each bit (MSB ‚Üí LSB):
    Copy these formulas across the row (change the divisor):
    • B2: =INT(MOD($A2/128,2))
    • C2: =INT(MOD($A2/64,2))
    • D2: =INT(MOD($A2/32,2))
    • E2: =INT(MOD($A2/16,2))
    • F2: =INT(MOD($A2/8,2))
    • G2: =INT(MOD($A2/4,2))
    • H2: =INT(MOD($A2/2,2))
    • I2: =INT(MOD($A2/1,2))
  3. Join the bits into one 8-bit string in J2:
    =TEXTJOIN("",TRUE,B2:I2)
  4. (Alternative with built-ins) In K2 you can also try:
    =DEC2BIN($A2,8)
    (Works if Engineering functions are available in your Excel.)
  5. Extension ‚Äì Hex with VLOOKUP:
    1. Make a small table on the side (e.g., M2:N17) with decimal 0‚Äì15 and hex 0‚ÄìF.
    2. Compute the two hex digits:
      • High nibble in L2: =INT($A2/16)
      • Low nibble in M2: =MOD($A2,16)
    3. Look them up to hex letters:
      • N2: =VLOOKUP(L2,$M$2:$N$17,2,TRUE)
      • O2: =VLOOKUP(M2,$M$2:$N$17,2,TRUE)
    4. Combine in P2: =N2 & O2

    Note: You could also use =DEC2HEX($A2,2) if available, but today we practise VLOOKUP logic.

  6. Format cells with a clear font and big size so it‚Äôs easy to read.
