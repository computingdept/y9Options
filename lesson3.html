What you’ll learn

By the end of this lesson, you will be able to:

  • Convert a decimal number (0–255) to 8-bit binary.
  • Convert a decimal number to hex (base 16).
  • Build an Excel sheet that converts numbers to 8-bit binary.

How you’ll know you’ve got it:

  • Your CS conversions match the correct binary/hex.
  • Your IT spreadsheet shows the right 8 bits for 0–255.
  • Extension: your sheet also shows the hex using VLOOKUP.

Key Vocabulary

Binary
Base-2 number system using 0 and 1 only.
Bit
A single binary digit (0 or 1).
Byte
8 bits. Can store numbers 0–255.
Hex (Hexadecimal)
Base-16 number system (0–9 then A–F).

CS Activity – Convert Decimal to 8-bit Binary and Hex

Work with numbers from 0 to 255. Convert each to 8-bit binary and to hex.

  1. Pick 5 numbers (e.g., 7, 13, 25, 128, 255).
  2. Write their binary using place values: 128 64 32 16 8 4 2 1.
  3. Write their hex (two digits, 00 to FF).
  4. Check your answers with a calculator or Excel (later).

IT Activity – Build an Excel Sheet: 0–255 → 8-bit Binary (Extension: Hex)

Create a spreadsheet that converts a decimal number to an 8-bit binary string. Then add an extension using VLOOKUP to show the hex value.

  1. In cell A2, type a number from 0 to 255 (try lots!).
  2. In cells B2:I2, show each bit (MSB → LSB):
    Copy these formulas across the row (change the divisor):
    • B2: =INT(MOD($A2/128,2))
    • C2: =INT(MOD($A2/64,2))
    • D2: =INT(MOD($A2/32,2))
    • E2: =INT(MOD($A2/16,2))
    • F2: =INT(MOD($A2/8,2))
    • G2: =INT(MOD($A2/4,2))
    • H2: =INT(MOD($A2/2,2))
    • I2: =INT(MOD($A2/1,2))
  3. Join the bits into one 8-bit string in J2:
    =TEXTJOIN("",TRUE,B2:I2)
  4. (Alternative with built-ins) In K2 you can also try:
    =DEC2BIN($A2,8)
    (Works if Engineering functions are available in your Excel.)
  5. Extension – Hex with VLOOKUP:
    1. Make a small table on the side (e.g., M2:N17) with decimal 0–15 and hex 0–F.
    2. Compute the two hex digits:
      • High nibble in L2: =INT($A2/16)
      • Low nibble in M2: =MOD($A2,16)
    3. Look them up to hex letters:
      • N2: =VLOOKUP(L2,$M$2:$N$17,2,TRUE)
      • O2: =VLOOKUP(M2,$M$2:$N$17,2,TRUE)
    4. Combine in P2: =N2 & O2

    Note: You could also use =DEC2HEX($A2,2) if available, but today we practise VLOOKUP logic.

  6. Format cells with a clear font and big size so it’s easy to read.
Cell Example for A2=25 Why
B2:I2 0 0 0 1 1 0 0 1 16+8+1 = 25 → bits at 16, 8, 1 are 1
J2 (TEXTJOIN) 00011001 Joined bits make the 8-bit string
L2 / M2 L2=1, M2=9 25÷16=1 remainder 9 → hex is 0x19
N2 / O2 (VLOOKUP) N2="1", O2="9" Lookup digits 1 and 9 in the 0–F table
P2 (hex) "19" Final hex = “19”
