Teacher Guidance – Lesson 5
Networks (CS) vs Sharing & Permissions (IT)

Overview

Approx. length: 60 minutes Year group: 9 Options Taster

Focus: CS — understand LAN basics and sketch a simple classroom network. IT — practise sharing files/folders with correct permissions in OneDrive/SharePoint.

Learning Objectives

  • Explain LAN and key devices (router, switch, AP).
  • Sketch a simple classroom network using the “roads” analogy.
  • Share files/folders with Can edit vs Can view correctly and verify behaviour.

Lesson Flow (approx.)

Students complete the Do Now silently in Microsoft OneNote (Class Notebook) under “Lesson 5 – Do Now”. Throughout the lesson, pupils must use the Snipping Tool (Win + Shift + S) to evidence their CS and IT work in OneNote.
Approx. Time Stage Teacher Guidance (modelling optional) Student Outcome / Evidence
Start (≈ 5 min) Do Now – Silent recap Display the Do Now. Remind: quiet answers in OneNote; reveal and check one or two. Responses saved under “Lesson 5 – Do Now”.
≈ 5 – 12 min Knowledge Recap Run through the two Knowledge boxes (LAN/devices; sharing/permissions) using the road/keys analogies. Class can name devices and describe view vs edit.
≈ 12 – 28 min CS – Design a classroom network If helpful: Model placing a switch centrally, adding devices, then router→internet. Mention private IPs and simple labels. Network diagram created in the 101Computing tool (or drawn). Evidence: Snip the finished design and paste into OneNote under “Lesson 5 – CS Activity”.
≈ 28 – 45 min IT – Share & set permissions If helpful: Model sharing one file as Can edit vs Can view and show how to test with a partner. Two files shared correctly (one editable, one read-only) and tested. Evidence: Snip the sharing dialog and partner test result into OneNote under “Lesson 5 – IT Activity”.
≈ 45 – 53 min Key Vocabulary (after practice) Reinforce: LAN, Router, Switch, AP, Permission, Share link, Owner. Ask: “Where did you use each today?” Verbal recall; optional brief note in OneNote.
≈ 53 – 60 min Exit Ticket / Reflection Use the three questions from the student page. Responses in OneNote. Saved under “Lesson 5 – Exit Ticket”.

Support & Extension

  • Support: Provide a device icon key; pre-labelled IP examples; step-by-step share walkthrough screenshots.
  • Extension: Add a wireless AP and two laptops; explain DHCP vs static IP in one sentence; set an expiry on a share link and record why.
  • SEND: Keep fonts large/high-contrast; chunk instructions; allow paired practice; modelling used only if helpful.
