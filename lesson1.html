Lesson 1 – The Digital World: Where Do Computer Science and IT Meet?

Do Now

Match each statement to Computer Science or IT.

1. Designs and builds programs.
Computer Science
2. Uses software to analyse business data.
IT
3. Understands algorithms and logic.
Computer Science
4. Creates digital graphics or presentations.
IT

What you’ll learn today

By the end of this lesson, you will be able to:

  • Explain what Computer Science and Information Technology (IT) mean.
  • Identify real examples of each in everyday life.
  • Describe where the two subjects overlap.

How you’ll know you’ve got it:

  • Correctly classify tasks as CS or IT and justify your choice.
  • Give a real-world example of a Computer Science and an IT job.
  • Summarise the difference in one clear sentence.

Key Vocabulary

Algorithm
A step-by-step procedure for solving a problem.
Program
A set of instructions that a computer executes.
Hardware
The physical parts of a computer system.
Software
The programs and applications that run on a computer.
Data
Information that a computer stores or processes.

Activities

Compare the focus of Computer Science and IT through two short practical tasks.

Computer Science Activity Information Technology Activity
Design an Algorithm
Write clear, step-by-step instructions for a simple task (e.g. making a jam sandwich). Test them with a partner and debug any mistakes. 		Create a Digital Document
Use presentation or word-processing software to create a short guide showing how technology supports daily life. Include text and images.

Reflection: Discuss how both activities involve problem-solving, but in different ways — one through building technology, the other through using it effectively.

Plenary

Cold-call: choose one task from today and explain whether it is CS, IT, or Overlap – and why.
