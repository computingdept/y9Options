Teacher Guidance – Lesson 4
Images & Colour (CS) vs Slide Design (IT)

Overview

Approx. length: 60 minutes Year group: 9 Options Taster

Focus: CS — how bitmaps store pixels/colour and how to estimate file size. IT — improve slide readability using clear design choices.

Learning Objectives

  • Explain pixels, resolution, colour depth.
  • Estimate file size with w × h × bpp ÷ 8.
  • Redesign slides for clarity (contrast, alignment, consistency).

Lesson Flow (approx.)

Students complete the Do Now silently in Microsoft OneNote (Class Notebook) under “Lesson 4 – Do Now”. Use the Snipping Tool (Win + Shift + S) later to evidence CS and IT work in OneNote.
Approx. Time Stage Teacher Guidance (modelling optional) Student Outcome / Evidence
Start (≈ 5 min) Do Now – Silent recap Display the Do Now (pixels/binary recap). Remind: quiet answers in OneNote; reveal briefly. Answers saved in OneNote under “Lesson 4 – Do Now”.
≈ 5 – 12 min Knowledge Recap Use Knowledge boxes: pixel/resolution/colour depth; file-size formula; slide design checklist. Analogy: mosaic tiles for pixels; shop window for slides. Class can state what changes file size and what makes slides readable.
≈ 12 – 28 min CS – Bitmap size & table task If helpful: Model one row: compute bits (w×h×bpp) then bytes (÷8). Clarify units (bits vs bytes). Point students to the 101Computing page. Table completed with correct sizes + two short notes (resolution ↑ or depth ↑ → size ↑). Evidence: Snip the completed table and paste in OneNote under “Lesson 4 – CS Activity”.
≈ 28 – 45 min IT – 3–4 slide deck from article If helpful: Model one slide title + one clear visual + 3 bullets; emphasise contrast and 24pt+. Keep text minimal; one message per slide. Short, readable deck explaining pixels/resolution/colour depth + one worked size example. Evidence: Snip slides (or export to PDF and snip first page) into OneNote under “Lesson 4 – IT Activity”.
≈ 45 – 53 min Key Vocabulary (after practice) Reinforce: Pixel, Resolution, Colour Depth, File Size, Contrast. Ask pupils to point to where each appears in their work. Verbal recall; optional brief note in OneNote.
≈ 53 – 60 min Exit Ticket / Reflection Use the four questions from the student page (pixel, formula, readability tip, reducing size). Responses in OneNote. Saved under “Lesson 4 – Exit Ticket” in Class Notebook.

Support & Extension

  • Support: Provide a mini file-size formula card; pre-fill a table template; allow paired reading of the article.
  • Extension: Compare BMP vs PNG vs JPEG (lossless/lossy); ask students to compress an image and report before/after sizes.
  • SEND: Keep fonts large/high-contrast; chunk instructions; model only if helpful; ensure independent use of Snipping Tool for evidence.
