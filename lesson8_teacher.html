Teacher Guidance – Lesson 8
Logic (CS) vs Digital Media (IT)

Overview

Approx. length: 60 minutes Year group: Y9 Options

Focus: CS — build truth tables (AND/OR/NOT) and apply to a simple rule. IT — produce a one-slide poster, export to PNG/JPEG, compare size/quality.

Learning Objectives

  • Complete truth tables for AND, OR, NOT and explain when each is true.
  • Apply logic to a short real scenario (e.g., door access).
  • Export media in two formats and justify the better choice.

Lesson Flow (approx.)

Students complete the Do Now silently in Microsoft OneNote (Class Notebook) under “Lesson 8 – Do Now”. Throughout the lesson, pupils must use the Snipping Tool (Win + Shift + S) to evidence both CS and IT tasks in OneNote.
Approx. Time Stage Teacher Guidance (modelling optional) Student Outcome / Evidence
Start (≈ 5 min) Do Now – Silent retrieval Display Do Now; remind “answers in OneNote, no discussion yet”. Brief reveal. Saved under “Lesson 8 – Do Now”.
≈ 5 – 10 min Set up today’s focus Define Boolean, AND/OR/NOT in one line each. Quick check with hands-up true/false. Class can state when AND/OR is true; what NOT does.
≈ 10 – 28 min CS – Truth tables + scenario If helpful: Model one row of AND and one row of OR. Use the lab-door scenario. Prompt: “What inputs? When is it true?” Completed tables and a one-line rule explanation. Evidence: Snip filled tables into OneNote “Lesson 8 – CS Activity”.
≈ 28 – 50 min IT – Poster & exports If helpful: Model a minimal slide (title ≤6 words, one icon, high contrast). Demonstrate Export → PNG and Export → JPEG; show file sizes. Slide exported to PNG & JPEG; sizes noted; one sentence comparing quality/size. Evidence: Snip the slide and file properties into OneNote “Lesson 8 – IT Activity”.
≈ 50 – 56 min Key Vocabulary (quick reinforce) Boolean, AND, OR, NOT, Truth table, Compression, Resolution, Format. Verbal recap or short OneNote note.
Final minutes Exit Ticket Use student page questions. OneNote submission. Saved under “Lesson 8 – Exit Ticket”.

Success Criteria

  • Truth tables: 4/4 rows correct for AND and OR; NOT flips correctly.
  • Scenario: clear statement of when the door opens (A∧B).
  • Poster: readable layout; exports compared with a justified choice (PNG for sharp text/logos, JPEG smaller for photos).

Support & Extension

  • Support: Provide partially filled tables; give the door rule verbally; offer sentence stems (“It is true when…”).
  • Extension (CS): Add a third input (e.g., A∧(B∨C)) and complete an 8-row table.
  • Extension (IT): Export at two resolutions and discuss how resolution vs format affects size/quality.
  • SEND: Large fonts/high contrast; chunk steps; modelling used only if needed; ensure independent Snipping Tool use for evidence.
