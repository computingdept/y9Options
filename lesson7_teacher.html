Teacher Guidance – Lesson 7
Operating Systems (CS) vs File Management (IT)

Overview

Approx. length: 60 minutes Year group: 9 Options Taster

Focus: CS — core OS responsibilities (UI, memory, process, file system). IT — tidy folder trees, smart filenames, search skills, and a simple backup plan.

Learning Objectives

  • Explain two or more OS jobs and match them to real problems.
  • Create a clear folder structure and apply a consistent naming convention.
  • Use search filters and plan a basic two-copy backup.

Lesson Flow (approx.)

Students complete the Do Now silently in Microsoft OneNote (Class Notebook) under “Lesson 7 – Do Now”. Throughout the lesson, pupils must use the Snipping Tool (Win + Shift + S) to evidence both CS and IT tasks in OneNote.
Approx. Time Stage Teacher Guidance (modelling optional) Student Outcome / Evidence
Start (≈ 5 min) Do Now – Silent retrieval Display Do Now. Quiet answers in OneNote; reveal briefly and move on. Saved under “Lesson 7 – Do Now”.
≈ 5 – 12 min Set up today’s focus Quickly define OS + four jobs (UI, memory, process, file system). Use one example per job. Verbal recall; readiness for matching task.
≈ 12 – 25 min CS – OS features matching If helpful: Model one item (“Two apps need RAM → memory management”). Prompt “Which OS part fixes this?” Completed matches + one-sentence justifications. Evidence: Snip the filled table/answers into OneNote under “Lesson 7 – CS Activity”.
≈ 25 – 48 min IT – File management If helpful: Model creating the folder tree, renaming a file to YYYY-MM-DD_topic_v1.ext, and a quick search (e.g., type:.pptx). Folder tree created; 5 files renamed/moved; search demonstrated; one-line backup plan. Evidence: Snip folder tree + a renamed file + search results into OneNote under “Lesson 7 – IT Activity”.
≈ 48 – 55 min Key Vocabulary (quick reinforce) Revisit: OS, UI, Process, Memory management, File system, Path, Extension, Metadata, Backup. Verbal check; optional short note in OneNote.
Final minutes Exit Ticket Use the three questions from the student page. Responses in OneNote. Saved under “Lesson 7 – Exit Ticket”.

Links & Resources

Support & Extension

  • Support: Provide a one-page OS cheat-sheet; sentence starters for justifications; template folder tree.
  • Extension: Add file tags (where supported); compare FAT/NTFS in one sentence; outline incremental vs full backups.
  • SEND: Large fonts/high contrast; chunk tasks; modelling only if helpful; ensure independent use of Snipping Tool for evidence.
