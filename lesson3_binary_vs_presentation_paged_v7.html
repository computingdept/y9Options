What you'll learn today

By the end of this lesson, you will be able to:

  • Explain what binary is and why computers use it.
  • Give examples of how text and images are represented.
  • Choose when to present data clearly for an audience.

How you'll know you've got it

  • Convert a small number to binary (and back).
  • Explain a simple encoding (e.g., characters or pixels).
  • Turn a small dataset into a clear chart.

Key Vocabulary

Binary — a number system that uses 0 and 1.
Bit / Byte — a single 0/1; eight bits make a byte.
Character set — a code that maps numbers to letters/symbols.
Pixel — a tiny dot of colour on a screen.
Chart — a picture that shows data clearly (bar, line, pie).

Main Activities

CS Track – Binary Basics

  1. Convert 13 to binary, then 10110 to decimal.
  2. Explain 1 sentence: why do computers use binary?
  3. Extension: what is a byte and a kilobyte?
# Example explanation:
# Computers use binary because their circuits have two stable states.

IT Track – Present the Data

  1. Enter a small dataset (e.g., favourite sports).
  2. Choose a suitable chart and add a title.
  3. Add labels so it’s clear and easy to read.

Compare & Reflect

  • When is a chart better than raw numbers?
