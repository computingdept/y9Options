What you’ll learn

  • What SQL (Structured Query Language) is used for.
  • How to use SELECT, WHERE, and ORDER BY to retrieve, filter, and sort data.
  • Difference between a query and results.
  • Apply SQL to solve practical problems on SQL Island.
  • (IT) Capture results clearly with screenshots and organise them on separate tabs/pages.

Key Vocabulary

SQLStructured Query Language for managing and querying data in relational databases.
QueryA request for data from the database.
SELECTRetrieves columns/rows from a table.
WHEREFilters rows based on a condition.
ORDER BYSorts the returned rows (e.g., ascending/descending).
TableA set of rows (records) and columns (fields) holding related data.
RecordA single row in a table.
FieldA single column (attribute) in a table.

Activities

Computer Science (Coding with SQL)

  1. Go to SQL Island.
  2. Work through at least 6–8 stages. Type your own SQL to progress.
  3. For each stage, write the SQL you used (e.g., SELECT * FROM citizens;) and a short note on what it does.
  4. Extension: Use AND, OR, and LIKE in your filters.
Starter snippets 
-- View everything in a table
SELECT * FROM citizens;

-- Filter (WHERE)
SELECT name, age FROM citizens WHERE age >= 18;

-- Sort (ORDER BY)
SELECT name, age FROM citizens WHERE age >= 18 ORDER BY age DESC;

Information Technology (Documenting Results)

  1. Open a blank document in Word or Google Docs.
  2. Create a separate tab/page for each SQL Island task you complete.
  3. After each task, use Snipping Tool (or macOS Screenshot) to capture:
    • Your SQL query area
    • The results output
  4. Label clearly: Task 1, Task 2, etc. Add a one‑line caption of what the query did.
  5. Save as SQL_Lesson6_YourName and submit via your usual platform.
Tips: Zoom to 110–125% for legibility; ensure the query and result both appear in the same snip.
