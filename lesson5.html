Knowledge Recap

üß† Computer Science ‚Äì Networks & Data Sharing

A network lets computers share data and resources. A router moves data between networks; a switch links devices inside one network.

  • LAN = one site (e.g., our classroom/school).
  • Router = traffic lights for data between roads (networks).
  • Switch = junction box that connects local devices.

ü™Ñ Analogy: Think of data as cars on roads. Houses = computers, roads = cables/Wi-Fi, switches = roundabouts, routers = traffic lights sending cars to other towns (the internet).

üóÇ IT ‚Äì Sharing & Permissions (OneDrive/SharePoint)

Sharing controls who can see a file and who can change it. Use Can edit for collaboration; use Can view for read-only.

  • Share a folder for teamwork so new files inherit the same permissions.
  • Use clear file names and versions (e.g., 2025-10-31_NetworkPoster_v1.pptx).
  • Remove access when the task is finished.

ü™Ñ Analogy: It‚Äôs like keys to a room: some people get a blue key (edit) and some a grey key (view only).

Key Vocabulary

LAN
Local Area Network (one site like a school building).
Router
Forwards data between different networks (e.g., school ‚Üî internet).
Switch
Connects devices inside a LAN and forwards frames.
Access Point (AP)
Lets wireless devices join the network (Wi-Fi).
Permission
What a person can do with a file/folder (view/edit).
Share link
A URL that grants access with rules (who, can view/edit, expiry).
Owner
The account that controls the file/folder and its permissions.
