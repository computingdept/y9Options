Key Vocabulary

Packet — a small chunk of data sent across a network.
IP address — a number that identifies a device on a network.
Router — a device that forwards data between networks.
Protocol — a set of rules for communication (e.g., HTTP).
Bandwidth — how much data can be sent per second.
Cloud storage — saving files on internet servers.
Permissions — who can view, comment, or edit a file.
Two‑Factor Authentication (2FA) — extra login step for security.
Phishing — a fake message that tries to steal info.
