Key Vocabulary

Operating System (OS)
Core software that runs the computer and apps.
User Interface (UI)
How you control the computer (windows, icons, menus).
Process
A program that is running.
Memory management
Shares RAM between programs safely.
File system
How files/folders are stored and found.
Path
The location of a file in folders.
Extension
The end of a filename that shows the type (e.g., .pptx).
Metadata
Extra info about a file (size, date, owner).
Backup
Another copy kept safe (e.g., cloud + local).
